## Installation and operation/安装与运行
```
# 下载到本地
git clone git@github.com:JackWong992/onlineMall.git

# 进入目录
cd onlineMall

# 安装依赖
npm install

# 服务端运行
npm run dev

# 显示
https://localhost:8080（windows->F12/MacOs->command+option+i => 点击手机调试，刷新页面 ）

# 项目打包
npm run build
```
